A simple front-end application generated from the cg-angular Yeoman
scaffolding generator.

To prepare a production version of the application:

cd frontend
bower install
npm install
grunt build

Production ready version of the app will be in the dist folder.

To develop using a local web server:

grunt serve

...and browse to <http://localhost:9001/index.html>

Technologies used:

Yeoman: <http://yeoman.io/>
cg-angular: <https://github.com/cgross/generator-cg-angular>
Bower: <http://bower.io/>
Grunt: <http://gruntjs.com/>
AngularJS: <https://angularjs.org/>
AngularUI: <https://angular-ui.github.io/>
Bootstrap: <http://getbootstrap.com/>
LESS: <http://lesscss.org/>

Development environment dependencies:
- nodejs: install from web site (not apt-get) <http://www.nodejs.org>
- bower
- grunt
